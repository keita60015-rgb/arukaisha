## About

株式会社虚実は、問いの製造・流通を事業とする架空の企業のホームページです。

パロディサイトですが、冗談でありつつも誰かを勇気づけ、前を向いてもらうことを目指しています。

## Philosophy

三つの思想が交差する場所に、虚実はあります。あるいは、ありません。

- **ストア哲学**（エピクテトス）— 制御できるのは自らの判断と意志のみ
- **曹洞禅**（道元・只管打坐）— ただ坐れ、意味を問うな
- **マルクス主義** — 労働の中に人間を取り戻せ

## Features

- 問いのラインアップ（入門〜根源まで3段階の深度）
- 名言千選（1000+の哲学的名言データベース）
- 「喝！！」オーバーレイ（臨済禅の目覚まし機能）
- 営業担当者の１日（99%の問い、1%のアウトプット）
- 創業から概念化までの沿革
