## About

ある合同会社は、問いの製造・流通を事業とする架空の会社のホームページです。

パロディサイトですが、冗談でありつつも誰かを勇気づけ、前を向いてもらうことを目指しています。

## Philosophy

「ある」——日本語で最も古く、最も単純で、最も哲学的な動詞。存在そのもの。

あなたが探しているものは、すでにある。

- **曹洞禅**（道元・現成公案）— 今ここに現れていること自体がすでに問い。修証一等。
- **ストア哲学**（エピクテトス）— 制御できるのは自らの判断と意志のみ
- **マルクス主義** — 労働の中に人間を取り戻せ

## Features

- 問いのラインアップ（入門〜根源まで3段階の深度）
- 名言千選（1000+の哲学的名言データベース）
- 「喝！！」オーバーレイ（臨済禅の目覚まし機能）
- 営業担当者の１日（99%の問い、1%のアウトプット）
- 創業から概念化までの沿革
